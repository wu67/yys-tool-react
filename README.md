## step by step

这个项目的只是我学习 react 的产物, 业务逻辑则是照搬了之前的个人项目 [yys-tool](https://github.com/wu67/yys-tool)

弃用 CRA / react-script, 已经一年没更新, 主要贡献者都在 t 上表露意向搞 SSR 了

#### install

```bash
npm i
```

#### start

```bash
npm run dev
```

#### build

```bash
npm run build
```

## use docker

在项目根目录运行命令即可启动

```bash
docker-compose up -d
```

### 停止

```bash
docker-compose stop
```
